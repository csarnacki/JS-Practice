JavaScript Condtionals and Functions

Creating a condition using the if and else keywords.

let num = 6;
if (num == 6) {
mystery_num = "The mystery number is six.";
} else {
mystery_num = "The mystery number is something else."; }

Creating a condition using the if, else and else if keywords.

let day = "Sunday";
if (day == "Monday") {
dayOfWeek = "It's the beginning of the week.";
} else if (day == "Tuesday") {
dayOfWeek = "It's still the beginning of the week but not as bad as Monday.";
} else if (day == "Wednesday") {
dayOfWeek = "It's the middle of the week.";
} else if (day == "Thursday") {
dayOfWeek = "It's getting close to the weekend.";
} else if (day == "Friday") {
dayOfWeek = "The weekend is nearly here! I can't wait!";
} else if (day == "Saturday") {
dayOfWeek = "Horray! The weekend is finally here! I'm so excited!";
} else if (day == "Sunday") {
dayOfWeek = "It's still the weekend but I have to go back to work tomorrow.";
} else {
dayOfWeek = "Please enter a valid day";
}

Creating a condition using a switch.

let temp = "hot";
switch (temp) {
case "hot":
weather = "It's really hot and sunny out here! Better grab some shorts and sunglasses.";
break;
case "warm":
weather = "It's pretty warm out today so I can wear a t shirt and shorts.";
break;
case "chilly":
weather = "Brrr! It sure is getting cold and windy out gere. I should put a jacket on.";
break;
case "cold":
weather = "It's freezing out here! I need s scarf and gloves to stay warm!";
break;
case "rainy":
weather = "I need to get an umbrella and rain boots if I want to go outside.";
break;
default:
weather = "I think I'll just stay inside today.";
}

Creating a function.

function favoriteDrink() {
return ("Root Beer");
}

Creating a function with a parameter and passing in a value.

function myPet(name) {
return ("I have a dog and her name is " + name + " and she is a black lab.");
}

Creating a function with a default parameter.

function greet(name = "Stranger") {
return ("Welcome " + name);
}
